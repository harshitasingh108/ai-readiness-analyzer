# AI Readiness Analyzer 🚀

A simple web app that evaluates how AI-friendly a website is.

## 🔍 Features
- Enter a website URL
- Get an AI readiness score (out of 100)
- View key issues affecting AI visibility

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS

## 🌐 Live Demo
https://ai-readiness-analyzer-chi.vercel.app

## 💡 How it works
This is a frontend-based mock tool. It simulates how AI systems evaluate websites based on structure, clarity, and content readiness.
